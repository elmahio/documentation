# Does elmah.io have an MCP server

Yes. elmah.io has an MCP (Model Context Protocol) server currently in beta, which you can access at: https://mcp.elmah.io/mcp

By connecting this server to AI-integrated tools like Claude, Cursor, Visual Studio, or VS Code, you can give your AI agent direct access to your log data. This allows the AI to analyze errors, identify trends, and suggest fixes based on your actual logs in real-time.
